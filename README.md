# Лабиринт (Labyrinth)

3D-игра-лабиринт в браузере на **Three.js** с 10 уровнями сложности, общим облачным рейтингом и поддержкой мобильных устройств.

**Играть онлайн:** [https://ayanzhexenbay.github.io/Labyrinth/](https://ayanzhexenbay.github.io/Labyrinth/)

---

## Описание

Вы управляете светящимся синим шаром и проходите лабиринт от старта до зелёного финиша. Каждый уровень сложнее предыдущего: лабиринт больше, скорость ниже. Очки начисляются за прохождение и скорость. Результаты всех игроков сохраняются в облаке и отображаются в общем рейтинге.

---

## Возможности

- 10 уровней с процедурной генерацией лабиринтов
- Ввод имени игрока перед началом
- Система очков и рейтинг (общий + по каждому уровню)
- Облачное сохранение через **Firebase Realtime Database**
- Управление с клавиатуры и виртуальный джойстик на телефоне
- Камера от третьего лица, освещение и тени
- Проверка столкновений со стенами
- Адаптивный интерфейс для ПК и мобильных браузеров

## Уровни

| № | Размер | Сложность |
|---|--------|-----------|
| 1 | 11×11 | Лёгкий |
| 2 | 13×13 | Простой |
| 3 | 15×15 | Средний |
| 4 | 17×17 | Сложнее |
| 5 | 19×19 | Сложный |
| 6 | 21×21 | Опытный |
| 7 | 23×23 | Мастер |
| 8 | 25×25 | Эксперт |
| 9 | 27×27 | Легенда |
| 10 | 29×29 | Босс |

Следующий уровень открывается после прохождения текущего.

---

## Система очков

- **Базовые очки:** уровень × 200
- **Бонус за скорость:** чем быстрее финиш, тем больше баллов
- В рейтинге учитывается **лучший** результат на каждом уровне
- В общем рейтинге колонка **«Пройдено»** показывает число фактически завершённых уровней (не открытых)

## Запуск локально

1. Склонируйте репозиторий:

```bash
git clone https://github.com/ayanzhexenbay/Labyrinth.git
cd Labyrinth
```

2. Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8080
```

3. Перейдите на [http://localhost:8080](http://localhost:8080)

---

## Настройка облачного рейтинга (Firebase)

Без Firebase игра работает локально в браузере. Для **общего рейтинга всех игроков** нужен Firebase:

1. Создайте проект на [Firebase Console](https://console.firebase.google.com)
2. Включите **Realtime Database** (тестовый режим)
3. Зарегистрируйте веб-приложение (`</>`) и скопируйте `firebaseConfig`
4. Вставьте данные в `firebase-config.js`
5. В **Realtime Database → Rules** установите:

```json
{
  "rules": {
    "leaderboard": {
      ".read": true,
      ".write": true
    }
  }
}
```

6. Залейте изменения на GitHub

В меню игры появится статус: **«Общий рейтинг онлайн»**.
